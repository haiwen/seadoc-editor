# Example App (Rspack)

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode with Rspack dev server.\
Default host/port are loaded from `.env.local` (`HOST`/`PORT`).

The page will reload when you make changes.\
Build/runtime errors are shown in the terminal/browser overlay.

### `npm test`

Runs unit tests via `jest`.

### `npm run build`

Builds the production bundle to the `build` folder via Rspack.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!
